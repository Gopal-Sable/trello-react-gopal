# Trello clone
- Demo link https://trello-react-gopal-snowy.vercel.app
## Functionalities 
- Board
    - Add board
    - Show boards
- List
    - add list
    - Show lists
    - Archive list
- Card
    - Add card
    - Show cards
    - check/uncheck card
    - Delete card
-Checklist
    - Add checklist
    - Show checklists
    - Delete checklist
- Check-Item
    - Add check-item
    - Show check-items
    - Delete check-item
    - check/uncheck check-Item
